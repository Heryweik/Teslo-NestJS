# Teslo API

1. Clonar proyecto
2. Instalar dependencias
```
yarn install
```
3. Clonar el archivo ___.env.template__ y renombrar a __.env__
4. cambiar las variables de entorno
5. Levantar la BD
```
docker-compose up -d
```
6. Ejecutar SEED
```
http://localhost:3000/api/sedd
```
7. Levantar: 
```
yarn start:dev
```
